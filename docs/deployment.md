# CashFlowCoin'u İnternete Açma

Bu proje Render Blueprint ile tek seferde şu bileşenleri oluşturur:

- CashFlowCoin web sitesi
- Spring Boot API
- PostgreSQL
- Redis uyumlu Key Value servisi

## Yayınlama

1. Değişiklikleri GitHub'a gönderin.
2. Render hesabınıza GitHub ile giriş yapın.
3. Render'da **New > Blueprint** seçin.
4. `Dybali/i2i-Academy-CryptoPal` deposunu bağlayın.
5. `GEMINI_API_KEY` alanına yeni Gemini anahtarınızı girin.
6. **Deploy Blueprint** düğmesine basın.
7. Tüm servisler yeşil olduğunda `cashflowcoin` statik sitesini açın.
8. Ekranın üst kısmındaki `onrender.com` adresi paylaşılabilir canlı bağlantıdır.

## Önemli

- API anahtarını GitHub'a göndermeyin.
- Ücretsiz backend 15 dakika kullanılmayınca uyur; ilk açılış yaklaşık bir dakika sürebilir.
- Ücretsiz Render PostgreSQL veritabanı 30 gün sonra sona erer. Daha uzun kullanım için veritabanını ücretli plana yükseltin.
